# RAG Engine v0.0.9

## Goal

Build a RAG engine from scratch to understand the architecture, not a
chatbot framework.

## Tech Stack

-   Runtime: Node.js (JavaScript)
-   Future API: Express (after v0.0.9)
-   LLM: OpenRouter (free models where available)
-   Embeddings: Hosted embedding API (initially Gemini or equivalent)
-   Vector DB: Qdrant (Docker)

## Principles

-   No LangChain/LlamaIndex.
-   Learn every pipeline step.
-   Modular design.
